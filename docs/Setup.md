# Setup

```bash
# Get project, example with git
git clone https://github.com/Viva-Resurs/Pixbo.git
cd Pixbo

# Back-end
composer install

# Front-end
npm install
```

#### Needed* tools

- [PHP](http://php.net/) *

- [Composer](https://getcomposer.org/download/) *

- [NodeJS & npm](https://nodejs.org/en/download/) *

- [Git](https://git-scm.com/download/win)
